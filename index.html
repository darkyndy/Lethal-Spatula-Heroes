HTML5 Game Jam with CraftyJS

This is a starter pack for 'platform' games.

To run this project, open index.html in a web browser.

Alternatively you can execute the run-server.sh script and navigate to http://localhost:8000 (Requires Python and bash)

Open sourced as part of the Mozilla Festival 2012's session 'HTML5 Mash Up with Mind Candy' For the slides to that sessions see here
